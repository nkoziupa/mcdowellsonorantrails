About

This site helps you plan McDowell Sonoran Trail routes.

How to Use the Site

On the main screen, select a trailhead or zoom in and select any map marker.

Select highlighted markers to create a route.

Select the current marker at any time to undo.

Select the info panel at any time to view your route plan or reset the map.

Tip: if you have only selected one marker and you have moved the map, tap the info panel to re-center the map on the marker.
